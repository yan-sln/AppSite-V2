Introduction

Tout bon programe contient une page d'aide très longue et très complète. Malheuresement ce n'est pas le cas de celui-ci, du moins pour cette version. Dès lors, nous aborderons ici seulement les raccourcis et les types d'erreurs.

1. Les raccourcis

    Ctrl+d : Apercu Web
    Ctrl+e : Exporter
    Ctrl+h : Aide
    Ctrl+n : Nouveau projet
    Ctrl+o : Ouvrir
    Ctrl+F11 : Plein écran
    Ctrl+F12 : Quitter
